If a number is divisible by 2, the least significant digit (rightmost)
will be an even number (2, 4, 6, 8 or 0)
37598 is divisible by 2
286409 is not divisible by 2

If all the digits in a number add up to another number that is divisible
by 3, the original number is divisible by 3
9036 is divisible by 3
The digits in 528 add up to 5 + 2 + 8 = 15 which is divisible by 3,
so 528 is also divisible by 3
The digits in 479 add up to 4 + 7 + 9 = 20, which is not evenly
divisible by 3, so 479 is not divisible by 3

If a number is divisible by 5, the least significant digit will
be a 0 or a 5

If a number is divisible by 11, the sum of the odd digits minus
the sum of the even digits will be 0 or a multiple of 11
The odd digits in 1067 add up to 7 + 0 = 7; the even digits
of 1067 add up to 6 + 1 = 7. Odd digits - even digits = 7 - 7 = 0,
so 1067 is divisible by 11
The odd digits of 8093624 add up to 4 + 6 + 9 + 8 = 27,
the even digits of 8093624 add up to 2 + 3 + 0 = 5.
Odd digits - even digits = 27 - 5 = 22 which is a multiple of 11,
so 8093624 is divisible by 11
Odd digits minus even digits of 1331 equal 0, so 1331 is divisible by 11

Best to always divide out prime factors in the same order, starting
with smaller numbers. A possible exception is dividing out 11s
before 7s, because it's usually easier to spot if a number is
divisible by 11 than by 7
